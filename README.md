"# my-online-shop" 

Це невеличкий pet-проєкт інтернет-магазину, який я створив з навчальною метою. Мета — потренуватися у створенні веб-додатку, попрактикувати роботу з backend, базою даних, авторизацією, та іншими речами.

Реалізовано:
- Реєстрація та авторизація користувачів
- Перегляд товарів
- Додавання товарів до бази (для продавців)
- Проста пошукова система
- Проста система профілів

Я розробив цей проєкт, щоб:
- Закріпити знання в C# та ASP.NET
- Попрактикувати SQL-запити та роботу з базою даних
- Створити перший більш-менш структурований проєкт
